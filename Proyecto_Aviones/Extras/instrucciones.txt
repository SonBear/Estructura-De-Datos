	             Instrucciones

1.  Para generar vuelos solo debe oprimir el boton generar, 
recuerde que este desaparecera una vez ingrese el numero 
de vuelos que necesite.

2. Si la pantalla no termina de imprimirse, o se queda con
espacios en blanco, puede presionar el boton actualizar, 
para volver a imprimir los aviones en pantalla.

3. Si presiona el boton Priorizar se activa una ventana para
eligir el numero de vuelo que necesite salir ahora.

4. El boton quitar sirve para eliminar el siguiente vuelo en
cola.
